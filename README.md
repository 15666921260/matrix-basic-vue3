# 项目初始化

## 环境

`node 18.16.0`

`npm  9.6.7`

## 框架

`Vue3`

`element-plus`

`ts`

## 框架初始化

### 1、`vite`构建项目

执行下面之前可能需要执行 `npm config set strict-ssl false`

`npm init vite`

### 2、路由

开发环境

`npm install vue-router@4 -S`

### 3、element-plus

```
npm install element-plus --save
```

### 4、按需自动导入插件

```
 npm install unplugin-vue-components -D
```
