# 项目初始化

## 环境

`node 18.16.0`

`npm  9.6.7`

## 框架

`Vue3`

`element-plus` 

## 框架初始化
### 1、安装vue3

使用vue3笔记初始化，vue3项目

### 2、安装Element plus

按照Element plus官网安装，以及引入

如果遇到问题：

warning Replace `'element-plus'` with `"element-plus";`

执行：`npm run lint --fix` 后重启

### 3、安装`axios`

前后端交互接口工具 `npm install axios` 可能需要开启vpn

### 1、`vite`构建项目

执行下面之前可能需要执行 `npm config set strict-ssl false`

`npm init vite`
